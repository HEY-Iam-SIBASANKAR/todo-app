\# Todo App



A fullstack Todo application built using:



\- Node.js

\- Express

\- MongoDB

\- jQuery

\- Bootstrap



\## Features



\- User Registration

\- Login Authentication

\- Add Appointments

\- Edit Appointments

\- Delete Appointments

\- Username availability check
