\# Todo App



A fullstack Todo application built using:



\- Node.js

\- Express

\- MongoDB

\- jQuery

\- Bootstrap



\## Features



\- User Registration

\- Login Authentication

\- Add Appointments

\- Edit Appointments

\- Delete Appointments

\- Username availability check



\## How to Run



1\. Start MongoDB

2\. Go to server folder

3\. Run: node api.js

4\. Open frontend with Live Server
